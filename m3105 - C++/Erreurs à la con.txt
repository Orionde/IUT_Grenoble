==Sur les inclusions
	Dans le main
		Mettre les include de tous les objets déclarés dans le main, on sait jamais
	Dans le .cpp
		Si fichier normal
			Uniquement l'include du .h
		Si template
			
	Dans le .h
		Si fichier normal
			Idem main
		Si template
